## 📖 Описание проекта

Приложение предназначено для управления каталогом фильмов, пользователями и подписками онлайн-кинотеатра.

### Основные возможности:
- 🎬 **Каталог фильмов** с постерами и описанием
- 🔍 **Фильтрация по жанрам**
- ⭐ **Избранное** с защитой от дублирования
- 👁 **История просмотров** с фиксацией даты и времени
- 💎 **Управление подпиской** с проверкой активности и расчётом дней
- 🎯 **Рекомендации** по наиболее просматриваемому жанру

## 🗄️ База данных

База данных **OnlineCinemaDB** состоит из 7 таблиц:

| Таблица | Назначение |
|:---|:---|
| **Genres** | Жанры фильмов |
| **Movies** | Фильмы (название, год, длительность, рейтинг, постер) |
| **Users** | Пользователи |
| **Subscriptions** | Тарифы подписок |
| **UserSubscriptions** | Подписки пользователей (дата начала/окончания) |
| **Favorites** | Избранные фильмы |
| **Views** | История просмотров |
| **Ratings** | Оценки фильмов |
